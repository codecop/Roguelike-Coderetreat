# Rogue-Runtime

This is a Python project.

## Install virtualenv & dependencies

```bash
pip3 install virtualenv
virtualenv -p python3 venv
source venv/bin/activate
pip3 install -r requirements.txt
```

## Run the tests

```bash
python3 -m unittest
```

## Run the UI

To start the application:

```bash
python3 main.py
```
